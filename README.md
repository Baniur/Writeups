# 📝 My write-ups

<p>This repository contains a curated collection of cybersecurity write-ups documenting my solutions, methodologies, and learning process across various hands-on training platforms.</p>

<p>Each write-up provides a structured analysis of the challenge, covering investigation steps, tools used, technical reasoning, and key findings. The goal of this collection is to reinforce practical skills in areas such as digital forensics, incident response, threat hunting, log analysis, network analysis, and system compromise investigation.</p>

<p>This repository serves both as a personal knowledge base and as a resource for others interested in improving their cybersecurity skills through practical, real-world scenarios. It emphasizes analytical thinking, clear documentation, and reproducible methodologies aligned with real blue team and defensive security workflows.</p>
